# Schedule maintenance mode
As a Standard Validator or Governing Validator, you can schedule maintenance for a node.

1. Go to **My Profile > Maintenance mode**.

<img src={StepOne} width={280} />

2. Specify the duration of the maintenance period in epochs. 
The valid duration is 3-144 epochs (~30 minutes to ~24 hours), where 1 epoch equals 200 blocks. You can choose from the valid start epoch, then decide on the end epoch.

<img src={StepTwo} width={416} />

3. After you submit the request, the system displays the status and duration of the maintenance period.

<img src={upcomingMaintenance} width={280} />
<img src={ongoingMaintenance} width={280} />

### Install a validator node 
These gudies assist individuals or institutions responsible
for setting up and confguring a validator node. 
On Ronin, validators are required to run a validator node *and* a bridge operator node.
The two methods of installation are via Docker or using the Ronin CLI. 

We recommended running both nodes on one machine:

<DocCardList items={[
    findSidebarItem('/docs/node-operators/mainnet/combined-validator'),

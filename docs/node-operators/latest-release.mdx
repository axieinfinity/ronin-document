# Upgrade to latest release

## Ronin node
Every Ronin upgrade is published on Ronin's GitHub: [(https://github.com/axieinfinity/ronin/releases)](https://github.com/axieinfinity/ronin/releases).

To find the Docker image for each release, visit these links:
* Mainnet:
    * Docker Hub: [https://hub.docker.com/r/axieinfinity/ronin-mainnet/tags](https://hub.docker.com/r/axieinfinity/ronin-mainnet/tags)
    * GitHub Container Registry: [https://github.com/axieinfinity/ronin/pkgs/container/ronin](https://github.com/axieinfinity/ronin/pkgs/container/ronin)
* Saigon testnet:
    * Docker Hub: [https://hub.docker.com/r/axieinfinity/ronin-testnet/tags](https://hub.docker.com/r/axieinfinity/ronin-testnet/tags)

### Latest image
Update `NODE_IMAGE` in your `env` file to the following version to download the latest image:
* `ghcr.io/axieinfinity/ronin:v2.5.2-9bf4895` or
* `axieinfinity/ronin-mainnet:v2.5.2-9bf4895`

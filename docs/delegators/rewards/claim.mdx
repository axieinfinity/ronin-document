# Claim rewards
Staking rewards do not automatically re-stake to your validators,
and instead need to be claimed manually.

1. Go to [RON staking](https://app.roninchain.com/staking) >
**My staking**.
2. Under **CLAIMABLE REWARDS**, select **Claim all**.
<img src={stepOne} width={1440} />

3. Select **Claim now**.
<img src={stepTwo} width={400} />

4. Confirm the transaction.
<img src={stepThree} width={400} />
